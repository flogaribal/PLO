The example sentences that will be generated are here meant to train the NLU model of a chatbot application used to book flights.

This file is the master file (meaning the program will be executed on it) and contains the intents. The first one will generate training and testing data, while the second one will only generate training data. We did two generations of the same intent but with different types of sentences, the second is a question or a request. More examples can be found on the file master.chatette.

The aliases are defined in the file `aliases.chatette` and slots are defined in files in sub-directory `slots/`. Both are included at the end of this file.
